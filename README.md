
### 1、标题
“#”（1-7）标题 表示1-7级标题，也可以用 # 标题 # 这种形式
# node
## node
### node

### 2、无序列表
前面用 “+” 或 “*” 或 “-”
+ 1
+ 2
+ 3

### 3、有序列表
直接用数学加小圆点
1. one
2. two

### 4、分割线
可以由"*" "-" "_" 这3个符号的至少3个
***
---
___
