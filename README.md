# 邮箱账号解封

* 线上项目url : https://aq.reg.163.com/rc/unlock  
* 使用webpack2.3.3 + regularjs（0.6.0-beta.2） 开发  
* 因项目接口少，直接在interface目录里进行接口模拟


## 开发指南

``` bash
# 依赖安装
  yarn

# 构建发布版本
  yarn dev

# 构建发布版本
  yarn build

```

## 目录结构说明

```
├── README.md
├── package.json
├── .gitignore
├── package.json
├── webpack  // 项目环境配置
├── dist  // 构建结果目录
├── interface  // 接口模拟数据
├── node_modules 
├── src  // 源码目录
│   ├── components  // 组件
│   ├── css  // css样式
│   ├── js  // 基本工具
│   ├── res  // 静态资源
│   ├── view  // 页面文件

```
